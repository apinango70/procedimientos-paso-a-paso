# Instalacion WSL en windows

## Listar distribuciones instaladas

```bash
wsl -l
```

## borrar distribucion

```bash
wsl --unregister <distroName>
```

## Listar distribuciones disponibles en linea para instalar

```bash
wsl -l -o
```

## Instalar distribucion

```bash
wsl --install -d <Distribution Name>
```
